    Class ControlsManager

    Manages player input controls, mapping raw input events (keyboard, gamepad, touch) to standardized game actions defined in ControlsConfig. Emits CONTROLS_EVENTS.PLAYER_ACTION events via the provided EventBus.

    getNormalizedMovementVector

    • getNormalizedMovementVector(): Point

      Calculates a normalized movement vector based on currently active movement actions. Assumes standard action names 'UP', 'DOWN', 'LEFT', 'RIGHT' are defined in the ControlsConfig actionMap. Useful for character movement logic.

      Returns Point

      A PixiJS Point object representing the normalized direction vector (magnitude 0 or 1). (0,0) if no movement, (1,0) right, (-1,0) left, (0,-1) up, (0,1) down. Diagonal movement is normalized to have a magnitude of approximately 1.
